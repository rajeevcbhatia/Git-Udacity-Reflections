When would you want to use a remote repository rather than keeping all your work local?

A. To create backups and share code

Why might you want to always pull changes manually rather than having Git automatically stay up-to-date with your remote repository?

A. Because I don't want to be in the middle of something and have Git pull incompatible code

Describe the differences between forks, clones, and branches. When would you use one instead of another?

A. fork - copy a repository to my account to make changes. clone - make a local copy of a repository. branch - make a separate copy locally to make changes and merge when required. this copy is not physically created but logically
